# homebrew-tap

Homebrew tap for [truth0530](https://github.com/truth0530)'s tools.

## Install

```sh
brew install truth0530/tap/screenpath
```

## Formulae

- **[screenpath](https://github.com/truth0530/screenpath)** — capture a screen region
  and copy the file _path_ to your clipboard (not the image), for terminal / tmux /
  AI-agent workflows on macOS.
